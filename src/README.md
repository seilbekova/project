## Project Overview
This project demonstrates core Object-Oriented Programming concepts in Java.
It includes two main classes: Student and Course, and a driver program Main.
The project applies encapsulation, abstraction, arrays of objects, and calculations.

## Class Descriptions

### Student Class
The Student class stores student information such as name, ID, major, GPA, and credits.
It uses private fields with public getters and setters.
Utility methods include GPA update, credit addition, and honors check.

### Course Class
The Course class represents a university course.
It contains an array of Student objects, demonstrating composition.
It calculates average GPA and identifies the student with the highest credits.
